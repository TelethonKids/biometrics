Template Instructions

These instructions can be disabled by changing the markdown front-matter to instructions: false.

By default, all assets (e.g. images, css style sheets and templates) are found in the assets directory and will need to be linked to using their path from the markdown document directory; the final HTML output is a self contained file.

Logo

The logo can be changed by updating the value of logo: new logo in the front-matter. The default logo is assets/images/logo800.jpg.

Navigation Bar

The table of contents can be displayed as a navigation bar or in traditional paper style. To display a sticky bootstrap navigation bar, both toc and toc-float should be set to true in the front-matter: i.e. toc: true and toc_float: true; the nav bar will only display the first level of headings. To display the headings as a more traditional table of contents change to toc_float: false. Setting toc: false will remove both navigation tools.

Header Themes

There are also several themes that that are based on Telethon Kids' style guide and website. The header background image and break-out box colours can be changed by replacing the appropriate lines of front-matter with the following code:

header-background: blue-triangles-darker

header-breakout: pms370-breakout

header-background: purple-dots-darker

header-breakout: pms144-breakout

header-background: green-circles-darker

header-breakout: pms2395-breakout

header-breakout: pms205-breakout
